Featured Project: Mastermind Board Game

An online version of the classic code-breaking game. The game was built using HTML, CSS, JavaScript, and jQuery. In the game, the player guesses at a randomly chosen secret color code until he/she either is correct, and wins, or makes 12 guesses, and loses. During each turn, the player receives feedback on how similar the guess was to the secret code. View live or on GitHub.

What I learned

From this project, I learned how to write mathematical algorithms in JavaScript to compare code guesses with the secret code. I also learned to manipuate state with jQuery and use CSS to create a game board displaying the correct colors.
